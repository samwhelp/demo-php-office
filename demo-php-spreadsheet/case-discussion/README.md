

## 安裝「phpoffice/phpspreadsheet」

執行

``` sh
$ composer install
```


## 如何測試


### hello

執行

``` sh
$ ./bin/hello.php
```

或是執行

``` sh
$ make hello
```

執行主要程式碼，請參考「[src/Demo/Hello.php](src/Demo/Hello.php)」。
